# Sales-dashboard
This Sales Dashboard is an interactive business intelligence project developed using Power BI to analyze and visualize sales performance. It transforms raw sales data into meaningful insights through dynamic charts, KPIs, and filters, enabling users to make data-driven decisions.
Key Features
Interactive dashboard with slicers and filters
Sales, Profit, and Revenue KPIs
Monthly and yearly sales trend analysis
Region-wise and category-wise performance
Top-selling products analysis
Customer and market insights
Easy-to-understand visualizations for business reporting
Tools & Technologies
Power BI
Microsoft Excel
Data Cleaning
Data Visualization
Business Intelligence (BI)
Project Objective

The objective of this project is to provide a clear overview of business performance, identify sales trends, monitor key metrics, and support informed decision-making through interactive visual reports.
